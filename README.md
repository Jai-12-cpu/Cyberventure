# 🎮 CyberVenture Game

Developed CyberVenture, a Python game with four levels and interactive sub-levels designed to challenge memory and problem-solving skills. Sub-levels include Number Guess, Unjumble, Game 24, and Cat Finder, each testing the player’s logic, calculation, or memory. The project aims to promote awareness about cyber safety and the consequences of neglecting good online practices.

## 🚀 Live Demo

## 📌 Project Overview

CyberVenture is a simple yet engaging game project designed to demonstrate:

- Frontend development using HTML  
- Basic game structure and interaction  
- Integration of logic through Python (for offline/local runs)  
- Use of MySQL for data storage and management  
- Deployment using GitHub Pages  

This project is ideal for beginners exploring game development, databases, and web hosting.

---

## 🛠️ Technologies Used

- HTML (for game interface)  
- Python (for logic and backend experimentation)  
- MySQL (for storing and managing game-related data)  
- GitHub Pages (for hosting the web version)

## ▶️ How to Run

### 🌐 Play Online
Simply open:  
https://jai-12-cpu.github.io/Cyberventure/

---

### 💻 Run Locally (Python + MySQL Version)

1. Install Python  
2. Install MySQL and start the server  
installation (Local Setup)
3. Clone the repository:
```
git clone https://github.com/Jai-12-cpu/Cyberventure.git
```
4. Navigate to the folder:
```
cd Cyberventure
```
5. Run Python scripts (if applicable):
```
python main.py
```
6. Open `index.html` in your browser
